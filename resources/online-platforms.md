# Online Mahjong Platforms

A comprehensive guide to playing Chinese Classical Mahjong online. This list includes platforms, features, and recommendations for players at all levels.

## Important Note

**Platform availability and features change frequently.** Always:
- Verify current availability before committing
- Check which rules/variants are supported
- Read recent reviews
- Test with free/trial options first

## Platforms Supporting Chinese Classical

### Mahjong Time

**Website:** mahjongtime.com

**Platforms:** Web browser (desktop/mobile)

**Variants:**
- Chinese Classical
- Hong Kong
- Taiwanese
- MCR (Modern Chinese Official)
- Many others

**Features:**
- ✓ Multiple rule sets
- ✓ Tutorial mode
- ✓ Play against AI
- ✓ Ranked and casual play
- ✓ Tournament support
- ✓ Statistics tracking
- ✓ Multi-language support

**Pros:**
- Authentic Chinese Classical rules
- Active player base
- Good for serious learning
- Desktop and mobile compatible
- Replay analysis available

**Cons:**
- Interface can be complex for beginners
- Some features require subscription
- Ads in free version
- Steeper learning curve

**Cost:** Free with ads; premium subscription for ad-free and extra features

**Best For:** Intermediate to advanced players, serious learners

**Beginner Friendly:** ⭐⭐⭐☆☆ (3/5)

### Real Mahjong

**Platform:** Mobile app (iOS/Android)

**Variants:**
- Chinese Classical
- Hong Kong Style
- Taiwanese
- Others

**Features:**
- ✓ Offline AI opponents
- ✓ Adjustable difficulty
- ✓ Clean, intuitive interface
- ✓ Tutorial included
- ✓ Rule customization
- ✓ Statistics
- ✓ No internet required (for AI play)

**Pros:**
- Great for offline practice
- Simple, clean design
- Beginner-friendly
- Customizable rules
- No pressure from live opponents

**Cons:**
- Limited online multiplayer
- Smaller player community
- AI can be predictable
- Fewer advanced features

**Cost:** Free with ads; paid version ad-free

**Best For:** Beginners, casual players, offline practice

**Beginner Friendly:** ⭐⭐⭐⭐⭐ (5/5)

**Platform:** Web, Mobile (iOS/Android), PC Client

**Variant:** **Japanese Riichi (NOT Chinese Classical!)**

**Why Listed Here:**
- Extremely popular and polished
- Many Chinese Classical players also play Riichi
- Good for understanding variant differences
- Can help appreciate Chinese Classical uniquely

**Features:**
- ✓ Beautiful anime-style graphics
- ✓ Excellent tutorial
- ✓ Active global community
- ✓ Ranked ladder
- ✓ Tournament support
- ✓ Detailed statistics

**Important:** Different rules from Chinese Classical! Learn separately.

**Best For:** Exploring Japanese variant, appreciating polished mahjong software

**Beginner Friendly (for Riichi):** ⭐⭐⭐⭐⭐ (5/5)

## Choosing the Right Platform

### For Complete Beginners

**Recommendation:** Real Mahjong (mobile)

**Why:**
- Start with AI opponents (no pressure)
- Learn at your own pace
- Clean, simple interface
- Offline practice anytime

**Progression:**
1. Week 1-2: Play against easy AI
2. Week 3-4: Increase difficulty
3. Month 2: Try online opponents
4. Month 3+: Explore other platforms

### For Serious Learners

**Recommendation:** Mahjong Time

**Why:**
- Authentic Chinese Classical rules
- Competitive community
- Statistics for tracking improvement
- Multiple variants to compare

**Approach:**
1. Complete tutorial mode
2. Play casual games to learn interface
3. Move to ranked when comfortable
4. Study replays to improve

### For Social Players

**Recommendation:** Mahjong Club

**Why:**
- Easy to find friends
- Social features enhance fun
- Regular events keep it fresh
- Mobile-first for convenience

**Usage:**
- Join clubs/communities
- Participate in tournaments
- Make friends who play
- Balance social fun with skill development

### For Hardcore Competitors

**Recommendation:** Mahjong Time + Tenhou (Riichi)

**Why:**
- Highest skill ceiling
- Serious player base
- Detailed statistics
- Tournament opportunities

**Note:** Competition scene is more developed for Riichi; Chinese Classical competition is more regional

## Platform-Specific Tips

### Mahjong Time

**Getting Started:**
1. Create free account
2. Complete tutorial (required)
3. Play "practice room" games first
4. Join beginner-friendly tables
5. Read rule descriptions carefully

**Tips:**
- Use "observer" mode to watch before playing
- Start with slower time limits
- Check which variant table uses before joining
- Save replays of interesting games

### Real Mahjong

**Getting Started:**
1. Download and install
2. Choose Chinese Classical in settings
3. Start with easiest AI
4. Complete a few games
5. Gradually increase difficulty

**Tips:**
- Use it for solo drills (see curriculum)
- Practice specific situations
- No shame in easy AI for learning
- Customize rules to match your group's preferences

### Mahjong Club

**Getting Started:**

## Dealing with Platform Issues

### Connection Problems

**If you disconnect:**
- Most platforms auto-play for you
- Try to reconnect immediately
- Apologize if you rejoin
- Check your internet before ranked games

**If others disconnect:**
- Be patient
- Don't assume intentional
- Report persistent disconnectors if needed

### Cheating/Suspicious Behavior

**If you suspect cheating:**
- Most platforms have reporting systems
- Save replay if available
- Don't accuse in chat
- Let moderators investigate
- Remember: unlikely in casual play, more common in high ranks

### Interface Bugs

**If something seems wrong:**
- Screenshot the issue
- Report through official channels
- Check FAQs (might be feature, not bug)
- Ask community if others experience it

**Recommendation:** Try both, use what feels best!

## Tracking Your Progress

### Platform Stats to Monitor

**Win Rate:**
- Target: 20-30% (normal for 4-player game)
- Below 20%: Focus on fundamentals
- Above 30%: You're doing great!

**Average Placement:**
- Target: 2.5 or better (out of 4)
- Track trend over time
- Better indicator than win rate alone

**Deal-In Rate:**
- How often you discard winning tiles
- Lower is better
- Indicates defensive skill

**Hand Value:**
- Average points per win
- Higher shows pattern mastery
- Balance with win rate (don't over-reach)
