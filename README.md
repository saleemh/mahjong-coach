# Chinese Classical Mahjong Learning Path

Welcome to your personal mahjong learning journey! This repository contains a structured curriculum to take you from complete beginner to confident casual player in Chinese Classical Mahjong.

## About This Learning Path

**Your Profile:**
- Learning variant: Chinese Classical Mahjong
- Experience level: Complete beginner
- Goal: Casual play with friends and family

**What You'll Achieve:**
By completing this curriculum, you'll be able to:
- Recognize all 144 tiles instantly
- Set up and manage a complete game
- Form valid winning hands consistently
- Calculate basic scores accurately
- Make strategic decisions about which tiles to discard
- Play confidently in casual social settings
- Teach the basics to another beginner

## How to Use This Repository

This curriculum is designed for self-paced learning over approximately 8 weeks, with ongoing refinement afterward. Each phase builds on the previous one, so it's recommended to follow them in order.

## Recommended Study Schedule

### Weeks 1-2: Foundation
- **Daily:** 30 minutes tile recognition and terminology
- **Study:** Complete all Phase 1 lessons
- **Goal:** Memorize all tiles and understand basic objective

### Weeks 3-4: Core Mechanics
- **Study:** Complete all Phase 2 lessons
- **Practice:** 2-3 practice games per week
- **Goal:** Play complete games (even if scoring is unclear)

### Weeks 5-8: Scoring & Strategy
- **Study:** Complete all Phase 3 lessons
- **Practice:** 3-4 games per week + 1 scoring exercise session
- **Goal:** Calculate scores and make strategic decisions

### Week 9+: Ongoing Refinement
- **Practice:** Regular play schedule
- **Review:** Analyze your games using the practice log
- **Explore:** Phase 4 advanced topics as needed

## Tips for Success

1. **Don't rush** - Mahjong has many details; take time to absorb each concept
2. **Practice regularly** - Consistent short sessions beat occasional long ones
3. **Use the practice log** - Tracking helps you see progress and identify weak areas
4. **Play online** - Great for practicing at your own pace
5. **Find a group** - Playing with others is the most enjoyable way to improve
6. **Ask questions** - Use the notes section to track things you're unsure about
7. **Have fun!** - Mahjong is a social game meant to be enjoyed
